# 📊 Alura Store – Análisis de Datos

## 🧠 Descripción del proyecto
Este proyecto corresponde al **primer challenge de Alura Latam – Análisis de Datos**.  
El objetivo es analizar los datos de ventas, rendimiento y reseñas de las 4 tiendas de **Alura Store** para ayudar al Sr. João a decidir **qué tienda debe vender** para iniciar un nuevo emprendimiento.

A través del análisis de datos y visualizaciones, se busca identificar la tienda **menos eficiente**, basándose en métricas clave de desempeño.

---

## 🎯 Objetivo del análisis
Evaluar y comparar las tiendas de Alura Store considerando:

- Ingresos totales
- Categorías más vendidas
- Productos más vendidos
- Calificaciones de clientes
- Costo promedio de envío

Con base en estos datos, se presenta una **recomendación final fundamentada**.

---

- Matplotlib
